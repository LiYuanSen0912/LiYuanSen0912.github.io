Button 按钮组件

按钮用于开始一个即时操作。

何时使用

标记了一个（或封装一组）操作命令，响应用户点击行为，触发相应的业务逻辑。

代码演示

基础用法

按钮有六种类型：默认按钮、主要按钮、次要按钮、成功按钮、警告按钮、错误按钮和信息按钮。

幽灵按钮

幽灵按钮将按钮的内容反色，背景变为透明，常用在有色背景上。

按钮尺寸

按钮有五种尺寸：极小、小、中等、大和超大。

按钮形状

按钮有三种形状：默认(方形)、圆角和圆形。

禁用状态

添加 disabled 属性即可让按钮处于不可用状态，同时按钮样式也会改变。

加载状态

添加 loading 属性即可让按钮处于加载状态，此时按钮不可点击。

块级按钮

block 属性将使按钮适合其父宽度。

链接按钮

提供 href 属性即可让按钮变成链接。

API

参数 说明 类型 默认值
variant 按钮类型 'default' | 'primary' | 'secondary' | 'success' | 'warning' | 'error' | 'info' 'default'
size 按钮尺寸 'xs' | 'sm' | 'md' | 'lg' | 'xl' 'md'
shape 按钮形状 'default' | 'round' | 'circle' 'default'
block 是否为块级元素 boolean false
ghost 是否为幽灵按钮（透明背景） boolean false
href 点击跳转的地址，指定此属性按钮的行为和 a 链接一致 string -
target 相当于 a 链接的 target 属性，href 存在时生效 string -
disabled 是否禁用状态 boolean false
loading 是否加载中状态 boolean false
type 设置 button 原生的 type 值 'button' | 'submit' | 'reset' 'button'
icon 设置按钮的图标组件 ReactNode -
onClick 点击按钮时的回调 (event) => void -
