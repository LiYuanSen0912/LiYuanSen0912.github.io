Vue

1. Vue生命周期

  • beforeCreate
  • created
  • beforeMount
  • mounted
  • beforeUpdate
  • updated
  • beforeDestroy
  • destroyed

2. Vue双向绑定原理

  • Vue.js 是采用数据劫持结合发布者-订阅者模式的方式，通过Object.defineProperty()来劫持各个属性的setter，getter，在数据变动时发布消息给订阅者，触发相应的监听回调。

3. Vue组件通信

  • 父子组件通信：props和$emit
  • 兄弟组件通信：$emit和$on
  • 跨级组件通信：$emit和$on、provide和inject、$attrs和$listeners
  • 全局状态管理：Vuex

4. Vue路由

  • Vue Router 是 Vue.js 官方的路由管理器。它和 Vue.js 的核心深度集成，让构建单页面应用变得易如反掌。
  • Vue Router 默认 hash 模式 —— 使用 URL 的 hash 来模拟一个完整的 URL，于是当 URL 改变时，页面不会重新加载。
  • Vue Router 支持路由嵌套、动态路由匹配、路由守卫等功能。

5. Vue虚拟DOM

  • 虚拟DOM是对真实DOM的抽象，它以JavaScript对象的形式保存在内存中，然后通过一系列操作使这个虚拟DOM和真实DOM保持同步。
  • 虚拟DOM的优点是：减少了对真实DOM的操作，提高了性能；可以跨平台操作，比如Web端、小程序端等。

6. Vue响应式原理

  • Vue.js 是采用数据劫持结合发布者-订阅者模式的方式，通过Object.defineProperty()来劫持各个属性的setter，getter，在数据变动时发布消息给订阅者，触发相应的监听回调。
  • Vue.js 2.x 中，使用 Object.defineProperty() 来劫持各个属性的 setter，getter，在数据变动时发布消息给订阅者，触发相应的监听回调。
  • Vue.js 3.x 中，使用 Proxy 来劫持各个属性的 setter，getter，在数据变动时发布消息给订阅者，触发相应的监听回调。
  • Vue.js 3.x 中，使用 Proxy 来劫持各个属性的 setter，getter，在数据变动时发布消息给订阅者，触发相应的监听回调。

7. Vue组件化

  • 组件化是Vue.js的核心思想之一，它允许我们将页面拆分成多个独立的、可复用的组件，从而提高代码的可维护性和可复用性。
  • Vue.js 提供了组件化的支持，包括组件的定义、注册、使用等。
  • Vue.js 组件可以包含模板、脚本、样式等，它们可以独立存在，也可以嵌套使用。

8. Vue指令

  • Vue.js 提供了一系列的指令，用于在模板中绑定数据、事件、样式等。
  • 常见的指令有：v-bind、v-model、v-for、v-if、v-show、v-on等。
  • v-bind：用于绑定属性，例如：v-bind:src=”imageSrc”。
  • v-model：用于双向绑定数据，例如：v-model=”message”。
  • v-for：用于循环渲染列表，例如：v-for=”item in items”。
  • v-if：用于条件渲染，例如：v-if=”show”。
  • v-show：用于条件渲染，例如：v-show=”show”。
  • v-on：用于绑定事件，例如：v-on:click=”handleClick”。
