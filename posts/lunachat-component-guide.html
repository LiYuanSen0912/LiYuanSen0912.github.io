LunaChat 聊天组件

LunaChat 是一个功能强大的 AI 聊天组件，支持多种大型语言模型，可以轻松集成到您的应用中，提供类似 ChatGPT 的对话体验。

何时使用

  • 需要在应用内集成 AI 对话功能
  • 希望提供多模型支持的对话界面
  • 需要管理多个对话历史的聊天应用
  • 提供基于 AI 的智能客服或助手功能

注意事项

  1. API 密钥安全：请确保在生产环境中不要将 API 密钥直接写在前端代码中，建议通过后端服务代理 API 请求。

  2. 响应式设计：LunaChat 组件支持响应式设计，在移动设备上会自动调整布局，隐藏侧边栏并通过抽屉菜单访问。

  3. 模型兼容性：不同的 AI 服务提供商有不同的模型名称和参数要求，请确保配置正确的 provider 和对应的模型选项。

  4. 本地存储限制：启用本地存储时，对话历史会保存在浏览器的 localStorage 中，有容量限制。对于大量对话，建议实现服务器端存储。

  5. 流式响应：组件默认启用流式响应，让 AI 生成的回复逐字显示，提供更好的用户体验。
