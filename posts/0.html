JavaScript

Web开发的核心编程语言

核心概念

JavaScript是一种高级的、解释型的编程语言，是Web开发的三大核心技术之一（HTML、CSS、JavaScript）。它可以在浏览器中运行，也可以在服务器端（Node.js）运行。

  • 动态类型、弱类型语言
  • 支持面向对象、函数式和过程式编程范式
  • 基于原型的继承机制
  • 异步编程模型（Promise、async/await）

关键特性

  • ES6+ 特性

    箭头函数、解构赋值、扩展运算符、class类、Promise等现代语法

  • DOM操作

    可以动态修改网页内容、结构和样式

  • 事件处理

    监听和响应用户交互（点击、滚动、键盘输入等）

  • AJAX/Fetch API

    无需刷新整个页面即可与服务器进行异步通信
