版本兼容性

组件库的版本兼容性信息：

组件库版本 React 版本 TypeScript 版本 styled-components 版本
1.x >=16.8.0 >=4.0.0 >=5.0.0
2.x >=17.0.0 >=4.3.0 >=5.3.0
3.x >=18.0.0 >=4.9.0 >=6.0.0

确保您的项目使用的相关依赖版本与组件库要求的版本兼容。

浏览器兼容性

组件库支持以下现代浏览器：

  • Chrome >= 60
  • Firefox >= 60
  • Safari >= 12
  • Edge >= 79
  • Opera >= 60

对于 IE11 和旧版浏览器，您需要添加相应的 polyfill。
